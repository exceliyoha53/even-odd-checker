# Even & Odd Number Checker

A simple Python program that checks whether numbers are even or odd
and saves the results to a text file.

## Features
- Accepts user input
- Uses a reusable function
- Saves output to a file
- Beginner-friendly and clean structure

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

```bash
python main.py
